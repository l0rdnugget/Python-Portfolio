A command line Python program that organizes a travel itinerary across three destinations.

What it does

Prompts the user to enter details for 3 travel destinations (city, country, duration, budget)
Validates that duration and budget are valid integers
Stores all destination data in a list of dictionaries
Displays a clean, structured travel itinerary

Concepts used

Input validation with while True and try/except
Data types and casting (int)
Lists and dictionaries
Formatted output
