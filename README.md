# Python Portfolio

A collection of Python projects built while learning

## Projects

### Grade Calculator
A program that takes student input including name and assignment scores, calculates the total and average score, and displays a formatted grade report.

### Travel Itinerary Planner
A program that organizes a travel itinerary across three destinations. Prompts the user for city, country, duration, and budget, then displays a structured itinerary.

### Weather Outfit Advisor
A program that suggests outfits based on current weather conditions. Takes temperature, weather type, and wind speed as input and recommends what to wear.



## Concepts Used:
- Variables and data types
- User input and output
- Basic arithmetic
- String formatting with f-strings
- User input and validation with try/except
- Selection statements (if/elif/else)
- String formatting with f-strings
- Functions
